Usability Evaluation of Georgia State University

Jane, Jeffrey, and Kenna

COGS 187A

1. Visibility of System Status

"The system should always keep users informed about what is going on, through appropriate feedback within reasonable time."
– Jakob Nielsen


When you click "Events" from the homepage, it leads you to a new page that doesn’t have the title "Events" anywhere. It just leads to an untitled page with the drop-down labels "Categories" and "Tags." The information on this page is related to events, but there’s no clear system display that the user is on the page they intended to click on.



When a user selects the heading "Giving," they are lead to a page that is titled "Development & Alumni Affairs." Nowhere on the page is there anything that confirms to the user that they have actually been lead to the page they intended to land at. This is misleading when the user was supposedly clicking on the “Giving” part of this website.



When the user goes to the “Contact Us” page, they receive immediate feedback from the navigation list on the left side of the page. The “about” navigation is emphasized in red, which tells the user that the contact page can be found directly under that navigation element. Contact Georgia State is also clearly written several times in order to emphasize to the user which page they are currently on. The visibility of system status is well executed here because the user is easily informed about the status of the page through appropriate feedback.
